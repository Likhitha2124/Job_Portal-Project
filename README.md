💼 Job Portal Web Application

📌 Project Overview

This is a Job Portal Web Application developed using HTML, CSS, JavaScript, Java, and MySQL.
The project allows users to view job listings, search for jobs, and navigate between different pages.

🚀 Features Implemented

*  Multi-page navigation (Home, Login, Apply, Profile)
*  Responsive and styled UI using CSS
*  Database connection using Java (JDBC)
*  Insert and fetch job data from MySQL
*  Basic login validation using Java

Technologies Used :
Frontend:
* HTML
* CSS
* JavaScript
Backend:
* Java (JDBC)
Database:
* MySQL
  
Conclusion :

This project demonstrates foundational skills in frontend development, Java programming, and database integration, making it a strong beginner-level full-stack project.
